# EduPath: ACPC Rank Predictor

## 📌 Overview
EduPath is a web-based application designed to help students predict their admission chances in Gujarat colleges based on their ACPC rank. By combining backend logic with a clean, responsive frontend, EduPath provides clarity and confidence to students navigating the admission process.

---

## 🚀 Features
- Rank Prediction Logic: Estimates admission chances based on ACPC rank and historical cutoffs  
- College Database Integration: Structured backend with categorized college data  
- Responsive UI: Built with HTML, CSS, JavaScript, and Tailwind CSS for accessibility across devices  
- User-Friendly Design: Simple input forms and clear results for maximum usability  
- Scalable Backend: Database structuring ensures easy updates with new cutoff data  

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript, Tailwind CSS  
- Backend: Node.js (or relevant framework)  
- Database: MongoDB Atlas (connected for structured college data)

## 📊 How It Works
1. Input Rank: Students enter their ACPC rank  
2. Prediction Engine: Backend logic compares rank with cutoff data  
3. Output: Displays possible colleges and courses with admission likelihood
